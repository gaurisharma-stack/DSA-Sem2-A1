# Algorithmic Efficiency & Recursion Toolkit (AERT)

This project is part of the Data Structures (Unit 1) assignment.

Assignment Overview:

• Data Structures and ADTs (what they are, why we use them, and common operations).

• Algorithmic Analysis (time/space complexity, Big-O/Big-Ω/Big-Θ, best/average/worst
cases).

• Algorithm Paradigms (Divide & Conquer, Greedy, Dynamic Programming) – only
conceptually.

• Recursion (base case, call stack, and how recursive solutions work).


#Main Task

Build: “Algorithmic Efficiency & Recursion Toolkit (AERT)” in Python

Your toolkit will have:

• One small ADT implementation (Stack ADT)

• Recursive solutions for Factorial and Fibonacci

• A recursive solution for Tower of Hanoi (with trace for N = 3)

• A recursive solution for Binary Search

• A short analysis report explaining complexity and paradigms
